<h3> <a name = "abundant"> abundant </a> </h3> 
<p>
<h4>
<i>Definition:</i> The number n is <i>abundant</i> if the sum of all its positive divisors except itself is more than n.
<p>
They are abundant above perfection, not to mention deficiency. See <a href = "#perfect"> perfect</a> and <a href = "#deficient"> deficient</a> numbers.
<p>
<i>First ten:</i> 12, 18, 20, 24, 30, 36, 40, 42, 48, 54
<p>There are 2487 abundant numbers below 10,000.
</h4>

<h3> <a name = "amicable"> amicable </a> </h3> 
<p>
<h4>
<i>Definition:</i> The number n is <i>amicable</i> if it belongs to an amicable pair. Two numbers n and m are called an <i>amicable pair</i> if the sum of all positive divisors of n is equal to the sum of all positive divisors of m and both are equal to n + m.
<p>
It all started with <a href = "#perfect"> perfect</a> numbers which are amicable with themselves. Those numbers adopted social virtues and qualities; for the parts of each of them have the power to generate the other. See also <a href = "#sociable"> sociable</a> numbers.
<p>
<i>First ten:</i> 220, 284, 1184, 1210, 2620, 2924, 5020, 5564, 6232, 6368
<p>There are 10 amicable numbers below 10,000.
</h4>

<h3> <a name = "automorphic(curious)">  automorphic(curious)</a> </h3> 
<p>
<h4>
<i>Definition:</i>The number n is called an <i>automorphic</i> number if (the decimal expansion of) n<sup>2</sup> ends with n. These numbers are also called <i>curious</i>.
<p>It is curious, how for a k-digit automorphic number n there is another automorphic number -- <span class="nowrap">10<sup>k</sup> + 1 - n</span>. For this to work with n=1, you have to treat 1 as a zero-digit number.
<p>

<p>There are 8 automorphic numbers below 10,000.
</h4>

<h3> <a name = "Carmichael"> Carmichael </a> </h3> 
<p>
<h4>
<i>Definition:</i> The composite integer n is a <i>Carmichael</i> number if <span class="nowrap">b<sup>n-1</sup> = 1 (mod n)</span> for every integer b 
which is relatively prime with n.
<p>
Carmichael numbers behave like prime numbers with respect to the most useful primality test, that is they pretend to be prime.
<p>
<i>First ten:</i> 561, 1105, 1729, 2465, 2821, 6601, 8911, 10585, 15841, 29341.
<p>There are 7 Carmichael numbers below 10,000.

<p>There are many ways Catalan numbers can be interpreted; there are <a href = "http://mathforum.org/advanced/robertd/catalan.html">some cool pictures here</a> and the <a href = "http://en.wikipedia.org/wiki/Catalan_numbers">Wikipedia article</a> is very good.

<i>First ten:</i> 1, 2, 5, 14, 42, 132, 429, 1430, 4862, 16796.
<p>There are 9 Catalan numbers below 10,000.

<h3> <a name = "sociable"> sociable </a> </h3>

<p> The 2-cycles are the <a href = "#amicable">amicable pairs</a> and the 1-cycles are the <a href = "#perfect">perfect numbers</a>. For some numbers it is very difficult to compute the aliquot sequence. The smallest number whose sequence has not been completely computed is 276.
<p>
<i>First ten (known):</i> 12496, 14264, 14288, 14316, 14536, 15472, 17716, 19116, 19916, 22744.
<p>There are no sociable numbers below 10,000.
</h4>

<h4><i>Definition:</i> The number n is called a <i>weird</i> number if it is <a href = "#abundant">abundant</a> but is not the sum of any subset of its proper factors.
<p>
<p>
<i>First ten:</i> 70, 836, 4030, 5830, 7192, 7912, 9272, 10430, 10570, 10792.
<p>There are 7 weird numbers below 10,000.
</h4>
